Introduction to the Data

Where you got it?

The data has been researched and taken from Gapminder. Since Internet has become the most essential component for all human beings, the research was focused on getting some interesting data on the same.

A brief description of what the data is trying to measure

The data represents individuals using the internet around the world. The data gives us Country Name on the y-axis and year starting from 1900 to 2019 on the x-axis. The data gives us individuals using the internet in these countries in terms of percentage of population. The data shows that there has been a huge rise in the individuals using internet from the year 1998 which signifies the rise of the technology era.

Plotting data

Now we will plot the data into the bar graph.

About the Graph

The graph depicts the percentage of individuals using internet in all the countries in a particular year.

About the vega-lite mark(s)

Mark Used:
1) Type: Value = bar, I used the Vega-lite plot bar for the data representation to best present data of all the countries in a particular year.
2) Fill: Value = Crimson, Brown, and Green. To make the bar gaph more visually appealing, I tried using different range of colors.

Adding Components

Button Component

I used the in-built component: the button to allow the user to click on a specific color (Crimson,Brown,Green) and change the color of the bar graphs.

Select Option Component

I used the in-built component: the select option to allow the user to click on a specific year (2014,2015,2016,2017,2018) and show the data for that particular year for all the countries.
